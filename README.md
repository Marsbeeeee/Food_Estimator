# Food Estimator 🍔🎮

Food Estimator 是一个基于前端的 **像素风餐厅卡路里与点餐体验 Demo**。  
项目通过 NES 风格 UI 和游戏化交互，让用户以“闯关”的方式浏览热门餐厅，并模拟点餐流程。

> 当前版本为前端 Demo，主要用于展示 UI 设计、页面跳转与交互动画。

---

## ✨ Features

- 🎮 NES.css 像素风界面设计
- 🏆 Trending 页面：餐厅排行榜（Podium + List）
- 🍗 餐厅详情页（以 KFC 为示例）
- 🧺 点餐动画：食物飞入餐盘
- 🗣 NPC 互动：不同食物触发不同对话
- 🎆 页面动画效果（火花、漂浮食物、角色动画）
