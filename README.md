# 🍽 FoodPilot

**AI 驱动的饮食热量与营养分析助手**

> 一个以 **自然语言交互为核心的 AI Agent 型 Web 应用**
> 帮助用户通过对话，快速理解食物的热量与营养结构，并获得可解释、可个性化的饮食建议。

---

## 🧭 项目简介

**FoodPilot** 是一个面向日常饮食场景的 AI Agent Web 应用。
用户只需使用自然语言描述食物或用餐情况，系统即可返回 **结构化、可解释的热量与营养分析结果**，并结合用户画像给出更合理的饮食建议。

本项目聚焦于：

* **可解释的营养估算**
* **连续对话驱动的用户认知**
* **Web + AI Agent 的完整交互闭环**

而非复杂或医疗级营养计算，适合作为 **AI Agent / Web + AI 融合方向** 的项目展示。

---

## ✨ 项目亮点

### 🤖 对话式 AI Agent 设计

* 支持自然语言输入食物或饮食场景
* AI 自动理解用户意图并输出结构化分析结果
* 以“解释型输出”替代黑盒数值

### 📊 清晰的营养拆解与热量估算

* 输出总热量（kcal）
* 拆解主要食材的能量来源
* 明确假设前提，降低误导性

### 🧠 基于用户画像的个性化建议

* 支持健康目标、热量目标与饮食偏好设置
* 后续对话中动态调整分析与建议逻辑
* 面向生活方式级建议，而非精准处方

### 📒 饮食历史记录（Food Log）

* 自动保存历史查询与分析结果
* 支持回看当时的对话上下文
* 强调“长期饮食认知”，而非一次性问答

### 🎨 温馨、克制的 UI 设计

* 健康与生活方式导向的视觉风格
* 减少技术压迫感，降低使用门槛
* 强调信息可读性与交互流畅度

## 🧩 核心功能模块

### 1️⃣ Ask FoodPilot（对话分析）

**功能说明**

用户输入任意食物或饮食描述，例如：

> “一份鸡肉沙拉，加半个牛油果”

**AI 输出内容**

* 热量估算（kcal）
* 食材级别的能量拆解
* 简要结论与后续饮食建议

---

### 2️⃣ My Food Log（饮食记录）

**功能说明**

* 自动保存用户的历史查询记录
* 支持回看对应的对话与分析结果
* 帮助用户形成连续的饮食认知，而非精确追踪

---

### 3️⃣ Profile（用户画像）

**可配置内容**

* 健康目标（减脂 / 增肌 / 健康 / 表现）
* 每日热量目标（kcal）
* 饮食偏好（如高蛋白、无特定限制等）
* 基础身体与生活方式信息（用于合理性估算）

所有设置仅用于 **生活方式级别的个性化建议**，不构成医疗或营养处方。

---

## 🧠 设计理念

* **解释优先**：让用户知道“为什么是这个数”
* **连续交互**：强调对话上下文，而非单次查询
* **Agent 思维**：AI 不只是回答问题，而是辅助决策
* **Web 友好**：以产品化视角构建完整体验

---
